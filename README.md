### 乱棋 - 特殊ルール将棋Webアプリ

普通の将棋に「相手の駒も動かせる」というルールを加えた、少し変わった将棋ゲームです。
通常の将棋では自分の駒だけを動かしますが、乱棋では相手の王将以外の駒も動かすことができます。相手の駒をあえて動かして邪魔したり、自分に有利な形に利用したりできるため、普通の将棋とは違った読み合いが生まれます。
将棋の基本的な駒の動きはそのままなので、将棋を知っている人ならすぐに遊べます。

**Tech:** Django / JavaScript / PostgreSQL / Docker / Nginx
